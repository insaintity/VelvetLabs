# Design System

The product should feel like a luxurious midnight record-production console.

Core traits:

- Deep black, navy and plum surfaces
- Rose, violet and blue illumination
- Editorial serif branding
- Dense but calm workspace layout
- Translucent panels with fine borders and subtle inner highlights
- Tabular numerals for time, durations, queue positions and usage

Primary CSS variables live in `app/globals.css`.

Avoid generic SaaS styling, bright white screens, oversized empty cards and uncontrolled neon.
